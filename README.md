## Rockford Reviewed Directory Application

A modern, headless business directory decoupling a lightning-fast Next.js (App Router) frontend from a secure WordPress backend. Engineered for Core Web Vitals and SEO, it utilizes WPGraphQL and Advanced Custom Fields (ACF) to serve dynamic business listings.

## Core Features:

Headless Authentication: JWT-secured login and Gravity Forms double-opt-in registration.

User Dashboards: Protected routes for users to manage their curated favorites and authored reviews.

Optimistic UI: Instant visual feedback on interactions (like favoriting a listing) backed by asynchronous Next.js Server Actions.

Centralized Data Layer: Fully customized WordPress backend acting strictly as a secure API vault.
